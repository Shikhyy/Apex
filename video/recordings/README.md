# Screen Recording Instructions

## Recording Setup

1. **Resolution:** 1920x1080
2. **Frame rate:** 30fps
3. **Duration:** ~3 minutes

## Recording Sequence

### Section 1: Landing Page (30 seconds)
- Navigate to localhost:3000
- Scroll through hero section
- Highlight key features

### Section 2: Dashboard (45 seconds)
- Click "Launch Dashboard"
- Show the live pipeline view
- Let it run for a few cycles

### Section 3: Agents Page (30 seconds)
- Navigate to /dashboard/agents
- Show agent cards with reputation scores
- Point out ERC-8004 scores

### Section 4: Veto Log (30 seconds)
- Navigate to /dashboard/veto-log
- Show blocked trades with reasons

### Section 5: Portfolio (30 seconds)
- Navigate to /dashboard/portfolio
- Show PnL chart and positions

## Export

- Export as MP4
- Place in `video/recordings/screen.mp4`
